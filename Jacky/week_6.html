# Response to feedback


In response to the user feedback that i got, ive decided to completely recreate the hi-fi mock up. I did this because there were some changes that i would like to make to the overall mock up, that would take too long for me to create/change.

In the new mockup, all the comments, issues, changes proposed were considered and included in the final design. I have also added a further step towards the end each “journey” that is required to be there but was missing due to a lack for foresight. But now its there in the new Hi-fi Prototype.

I’ve also decided to remove some elements of the register process to cut down on some features to allow for the user to get to use the app as soon as possible.

The process of re-creating the prototype was far more simpler because I knew how to uses Adobe XD with more proficiency making this possible. If not the prototype would feel very bloated with unnecessary elements that weren't used or had no purpose.
